# Code of Conduct

## Our standards
We expect respectful, constructive collaboration.

Examples of positive behavior:
- Being respectful and patient
- Giving and receiving feedback gracefully
- Focusing on what is best for the community

Examples of unacceptable behavior:
- Harassment, insults, or personal attacks
- Publishing private information without consent
- Disruptive or exclusionary conduct

## Enforcement
Project maintainers are responsible for clarifying and enforcing this code of conduct.

Report concerns through private channels described in `SUPPORT.md`.
